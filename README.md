# <p align="center">Conditional-GAN</p>
<hr>

GAN variant in which both the Generator and the Discriminator are conditioned on auxiliary data such as a class label during Training.

### Requirements

* torch = 1.6.0
* torchvision = 0.7
* PIL
* numpy
* pandas
* matplotlib
* opencv
